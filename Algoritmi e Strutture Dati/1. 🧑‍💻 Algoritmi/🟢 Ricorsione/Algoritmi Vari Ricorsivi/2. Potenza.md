passati come argomenti della funzione.

# Pseudocodice ðŸ§¬
``` Pseudocodice TI:"Potenza" "FOLD"
int Potenza(n, e)
	if e == 0:
		return 1
	else
		r = n * Potenza(e-1)
		return r
```
