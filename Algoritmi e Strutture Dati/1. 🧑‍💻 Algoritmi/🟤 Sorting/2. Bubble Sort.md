# Descrizione dell'Algoritmo üìÉ
Il Bubble Sort √® un¬†#algoritmo basato sulla ripetizione di un procedimento fondamentale. 
- La singola iterazione dell'algoritmo prevede che gli elementi dell'array siano #confrontati a **due a due**, procedendo in un verso stabilito (che si scorra l'array a partire dall'inizio in avanti, o a partire dal fondo all'indietro, √® irrilevante; d'ora in poi ipotizzeremo che lo si scorra partendo dall'inizio).
# Pseudocodice üß¨
``` Pseudocodice TI:"Bubble Sort" "FOLD"
void Bubble_Sort(A[])
	for i = 1 to n:
		for j = 1 to n-i:
			if A[j] > A[j+1]:
				temp = A[j]
				A[j] = A[j+1]
				A[j+1] = temp
```

# Complessit√† dell'algoritmo üî¨
- #### Tempi per Stringa
Line | Istruzione | Comando | Costo Istruzione
----- | ----- | ----- | -----
2 | for i = 1 to n | $c_1$ | n
3 | for j = 1 to n-i-1 | $c_2$ | $\sum\limits_{i = 1}^{n} {n-i}$
4 | if A[j] > A[j+1] | $c_3$ | $\sum\limits_{i = 1}^{n} {n-i}$
5 | temp = A[j] | $c_4$ | $true_{if}$
6 | A[j] = A[j+1] | $c_5$ | $true_{if}$
7 | A[j+1] = temp | $c_6$ | $true_{if}$

***
- #### Tempo di Esecuzione ‚åõ
$T(n)$ = $c$ + $2c ¬∑ (\sum\limits_{i = 1}^{n} {n-i})$ + $3c ¬∑ true_{if}$
***
- #### Caso Migliore üòÉ
`L'array √® gi√† ordinato` $\Rightarrow$ $true_{if} = 0$
 $T_{migliore}(n)$ = $c$ + $2c ¬∑ (\sum\limits_{i = 1}^{n} {n-i})$ ‚Åì $n^2$ $\Rightarrow$ $Œ©(n^2)$
 
- #### Caso Peggiore üò±
`L'array √® ordinato al contrario` $\Rightarrow$ $true_{if}$ = $\sum\limits_{i = 1}^{n} {n-i}$
$T_{peggiore}(n)$ = $c$ + $5c ¬∑ (\sum\limits_{i = 1}^{n} {n-i})$ ‚Åì $n^2$ $\Rightarrow$ $O(n^2)$
> [!Summary]
> - ***Caso migliore:*** $Œ©(n^2)$
> array gi√† ordinato
> -  ***Caso medio***: $Œ∏(n^2)$
> array disordinato casualmente
> - ***Caso peggiore***: $O(n^2)$
> array ordinato al contrario

# Codice dell'Algoritmo üêç

```PYTHON TI:"Bubble Sort" "FOLD"
def Bubble_Sort(A):
	n = len(A)
	for i in range(n):
		for j in range(n-i-1):
			if A[j] > A[j+1]:
				temp = A[j]
				A[j] = A[j+1]
				A[j+1] = temp
```
