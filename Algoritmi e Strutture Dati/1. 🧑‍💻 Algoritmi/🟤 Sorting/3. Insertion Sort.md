Per #ordinare un array di dimensione $n$ in ordine crescente con Insertion Sort si deve:
- Scorrere l'indice $i$ da $2$ a $n$;
- Confronta l'elemento corrente ($key$) con il suo predecessore;
- Se l'elemento $key$ Ã¨ piÃ¹ piccolo del suo predecessore, confrontalo con gli elementi precedenti.Â Sposta gli elementi piÃ¹ grandi di una posizione verso l'alto per fare spazio all'elemento scambiato;
# Pseudocodice ğŸ§¬
``` Pseudocodice TI:"Insertion Sort" "FOLD"
void Insertion_Sort(A[])                   
	for i = 2 to n:
		key = A[i]
		j = i-1
		while (j >= 1) and (key < A[j]):
			A[j+1] = A[j]
			j--
		A[j+1] = key
```

```PYTHON TI:"Insertion Sort" "FOLD"
def Insertion_Sort(A):
	n = len(A)

		key = A[i]
		j = i-1

			A[j+1] = A[j]
			j -= 1
		A[j+1] = key
```
