# Definizione
Il #limite_asintotico superiore si indica con $O$.
Date due funzioni $f(n)$ e $g(n)$, si dice che $f(n)=O(g(n))$ se $∃c>0$ tale che:
$$0≤f(n) ≤ c·g(n), ∀ n ≥ n_0$$
>$\lim_{n \to ∞} {\dfrac{f(n)}{g(n)}} = 0 \quad\Longrightarrow\quad f(n)=O(g(n))$
